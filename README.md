Sales Performance Analysis Dashboard (Excel)

→ Project Overview
This project analyzes large-scale food delivery sales data to evaluate overall sales performance, identify high-revenue regions and categories, and uncover trends across time, location, and customer preferences. The dashboard helps stakeholders understand business performance and make data-driven decisions.

→ Tools Used

Microsoft Excel

Pivot Tables & Pivot Charts

Power Query

Excel Functions & Calculated Fields

Slicers & Interactive Filters

→ Key Insights

Analyzed 197K+ sales records with total revenue exceeding ₹53M

Karnataka, Uttar Pradesh, Telangana, Maharashtra, and Delhi are the top revenue-contributing states

Recommended and Main Course categories generate the highest sales

Consistent customer satisfaction with an average rating of ~4.34

Quarterly and weekly trends highlight peak ordering periods

→ Data Cleaning & Transformation

Removed null and inconsistent values

Standardized date, state, city, and category fields

Created time-based columns (Quarter, Week, Day) for trend analysis

Ensured consistent pricing and rating formats

Structured raw data for efficient pivot-based analysis

→ Dashboard Features

Total sales, order count, and average rating KPIs

Sales breakdown by state, city, category, and food type

Time-series analysis using weekly and quarterly trends

Interactive slicers for dynamic filtering and exploration
